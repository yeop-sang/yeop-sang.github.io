# [Divisibility by 11](https://acmicpc.net/problem/6976)

## 문제

양의 정수를 입력으로 받고 정수가 11로 나누어 떨어지는지 여부를 확인하기 위해 아래에 설명된 알고리즘을 사용하여 검사하는 프로그램을 작성하십시오. 11로 나눌 수 있는 이 특정 테스트는 1897년 Charles
L. Dodgson(Lewis Carroll ).

연산:

테스트 중인 숫자가 두 자리 이상인 경우 다음을 수행하여 새 숫자를 만듭니다.

단위 자릿수 삭제 단축된 숫자에서 삭제된 숫자 빼기 나머지 숫자는 원래 숫자가 11의 배수인 경우에만 11의 배수입니다.

## 입력

평소와 같이 입력의 첫 번째 숫자는 뒤에 오는 양의 정수의 수를 나타냅니다. 각 양의 정수는 최대 50자리입니다. 양의 정수에는 선행 0이 없다고 가정할 수 있습니다.

## 출력

입력에 있는 각 양의 정수에 대해 출력은 숫자를 삭제하고 빼면서 형성된 일련의 숫자로 구성되며, 그 뒤에 원래 숫자가 11로 나누어 떨어지는지 여부를 나타내는 메시지가 표시됩니다. 다른 양의 정수에 대한 출력은
공백으로 구분됩니다. 윤곽.
